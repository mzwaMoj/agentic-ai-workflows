## Tips for Demo Testing

1. **SQL Questions**: Test the agent's ability to generate correct SQL queries and visualizations
2. **Chart Variety**: Try different chart types (bar, line, pie, scatter, histogram)
3. **Multi-Chart**: Request multiple charts in one question
4. **Product Questions**: Test RAG retrieval for product information
5. **Complex Queries**: Combine multiple dimensions and filters
